# Cyber-Socratic Arena — Архитектор Смыслов

Веб-приложение для учеников и учителей, которое учит осознанно работать с ИИ через сократовский метод: ИИ не даёт готовых ответов, а ведёт ученика через наводящие вопросы.

Построено на основе документа `App's_Architecture.md`.

## Стек

- **React 18** + **TypeScript** + **Vite**
- **React Router v6** — роутинг
- **Framer Motion** — анимации
- **Context API** — стейт-менеджмент (включая авторизацию)
- Мок-авторизация через `localStorage` (реальный бэкенд не нужен)

## Установка и запуск

```bash
npm install
npm run dev
```

Откроется `http://localhost:5173`.

```bash
npm run build    # production-сборка

```

## Переключение между аккаунтами

В правом верхнем углу — аватар текущего пользователя. Клик открывает выпадающее меню со всеми демо-аккаунтами — одним кликом можно переключиться между учениками и учителем, без повторного ввода пароля. Также есть кнопка «Выйти».

## Структура маршрутов

| URL              | Доступ            | Описание                            |
| ---------------- | ----------------- | ----------------------------------- |
| `/login`         | публичный         | Авторизация                         |
| `/student`       | только `student`  | Карта Прогрессивного Квеста         |
| `/student/chat`  | только `student`  | Сократовский чат с ИИ-наставником   |
| `/teacher`       | только `teacher`  | Аналитический дашборд учителя       |

При попытке зайти не в свою роль — редирект на свою.

## Ключевые фичи (по архитектуре)

- **Барьер «Что я знаю»** — перед первым сообщением ученик обязан описать, что уже знает по теме. Только после этого активируется чат.
- **Сократовский ИИ** — локальная симуляция наставника: отвечает наводящими вопросами, не выдаёт готовые ответы, подбадривает, ведёт к самостоятельному пониманию. Логика в `src/utils/socraticAI.ts`.
- **Финальная рефлексия** — после 3+ сообщений разблокируется шаг рефлексии. ИИ даёт доброжелательный фидбэк на основе глубины ответа.
- **Карта квеста** — 15 уровней, «туман войны» на закрытых, пульсация на активном, бейдж на пройденных.
- **Дашборд учителя** — метрики класса, таблица-рейтинг, прогресс-бары `promptQuality` и `thinkingIndependence`, топ-3 проблемных тем, персональные рекомендации.

## Дизайн-система

Следует гайдлайнам из документа:

- **Фон**: `#0D1117` (глубокий космос) + `#1A1D2E` (грифельный)
- **Акценты**: градиент `#6C63FF → #00D4FF`
- **Успех**: неоновый `#00E5A0`
- **Магия/награды**: `#A855F7 → #EC4899`
- **Ошибки**: мягкий оранжевый `#F97316` (не агрессивный красный)
- Скруглённые формы, шрифт Inter, неоновые свечения и плавные Framer Motion анимации.

## Замечания

- Весь стейт и прогресс — мок-данные. При «переключении» аккаунта учителя вы увидите реальные данные всех 4 учеников.
- Чат-ИИ — локальная эвристика, для демонстрации сократовского подхода. В бою это `POST /api/chat` через прокси-слой (см. архитектурный документ, раздел 3.2).
